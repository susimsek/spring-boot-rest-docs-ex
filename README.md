## Prerequisites

* Java 11
* Maven 3.3+

## Installation

```sh
mvn clean package
```

```sh
cd target
```

```sh
java -jar restdocs-example-0.0.1-SNAPSHOT.jar
```

## Used Technologies

* Spring Boot 2.4.0
* H2
* Spring Restdocs
* Spring Boot Web
* Spring Boot Validation
* Spring Boot Jpa
* Spring Boot Actuator
* Mapstruct
* Lombok
* Dev Tools
* Spring Boot Test
